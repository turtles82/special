📥 Installation Guide

Method 1: Drag & Drop (Easiest)

  1. Drag the button below into your bookmarks bar
  2. Open any Edpuzzle assignment
  3. Click the bookmark to activate

Method 2: Manual

  1. Right-click on the button above → "Bookmark link"
  2. Or create a new bookmark with this code:
    javascript: fetch("{{ script_js }}").then(r => r.text()).then(r => eval(r))

Requirements

  • Modern web browser (Chrome, Firefox, Edge, Safari)
  • CSP-disabling extension: Chrome/Edge or Firefox
  • Works on Canvas and Schoology-embedded Edpuzzles
